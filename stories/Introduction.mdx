# Premium React Loaders

Welcome to **Premium React Loaders** - a comprehensive collection of production-ready loading components for React applications.

## Features

- **28 Premium Components** across 7 categories
- **Full TypeScript Support** with exported type definitions
- **Zero Configuration** - no Tailwind or build setup required
- **Tree-shakeable** - only bundle what you use
- **Accessible** - built with ARIA labels and best practices
- **Performant** - hardware-accelerated CSS animations
- **Customizable** - multiple ways to style components

## Component Categories

### Skeleton Loaders (9 components)
Placeholder components that mimic content layout while loading:
- Skeleton (base)
- SkeletonText
- SkeletonAvatar
- SkeletonImage
- SkeletonCard
- SkeletonForm
- SkeletonList
- SkeletonTable
- SkeletonPage

### Spinner Loaders (7 components)
Rotating and animated spinners:
- SpinnerCircle
- SpinnerRing
- SpinnerDots
- SpinnerBars
- SpinnerGrid
- SpinnerWave
- SpinnerPulse

### Progress Loaders (4 components)
Progress indicators (determinate/indeterminate):
- ProgressBar
- ProgressCircle
- ProgressRing
- ProgressSteps

### Pulse Loaders (4 components)
Bouncing, pulsing, wave animations:
- PulseDots
- PulseWave
- PulseBars
- TypingIndicator

### Button Loaders (1 component) ✨ New in v2.2.0
Specialized loaders for button loading states:
- ButtonSpinner (with circle, dots, bars variants)

### Status Indicators (2 components) ✨ New in v2.2.0
Animated success and error indicators:
- SuccessCheckmark
- ErrorIndicator

### Overlay Loaders (1 component)
Full-screen or container-relative loader overlays:
- LoaderOverlay

## Installation

```bash
npm install premium-react-loaders
```

### Prerequisites

This library only requires React:

```bash
npm install react react-dom
```

> **Note:** v2.0.0+ no longer requires Tailwind CSS!

### Setup

Import the styles in your main entry file (e.g., `main.tsx` or `App.tsx`):

```typescript
import 'premium-react-loaders/styles';
```

That's it! No Tailwind configuration or additional setup needed.

## Quick Start

```typescript
import { Skeleton, SpinnerCircle, ProgressBar } from 'premium-react-loaders';

function App() {
  return (
    <div>
      {/* Skeleton Loader */}
      <Skeleton width={200} height={20} />

      {/* Spinner */}
      <SpinnerCircle size={40} color="#3b82f6" />

      {/* Progress Bar */}
      <ProgressBar value={75} showValue />
    </div>
  );
}
```

## Customization

All components support multiple customization methods:

- **className prop** - Pass custom CSS classes
- **style prop** - Inline CSS styles
- **color props** - Direct color control
- **CSS variables** - Global theming
- **ThemeProvider** - App-wide theme configuration (v2.1.0+)

Example:

```typescript
<SpinnerCircle
  size={48}
  color="#3b82f6"
  className="my-4"
  style={{ opacity: 0.8 }}
/>
```

With ThemeProvider (v2.1.0+):

```typescript
import { ThemeProvider } from 'premium-react-loaders';

<ThemeProvider theme={{ primaryColor: '#8b5cf6', defaultSize: 'lg' }}>
  <App />
</ThemeProvider>
```

## TypeScript

Full TypeScript support with exported types:

```typescript
import type { SkeletonProps, SpinnerCircleProps } from 'premium-react-loaders';

const MyComponent: React.FC = () => {
  const skeletonProps: SkeletonProps = {
    width: 200,
    height: 20,
    animate: true,
  };

  return <Skeleton {...skeletonProps} />;
};
```

## Performance

All components use CSS animations (not JavaScript) for optimal performance:
- Hardware acceleration with `transform` and `opacity`
- 60fps smooth animations
- Minimal bundle size
- Tree-shakeable exports
