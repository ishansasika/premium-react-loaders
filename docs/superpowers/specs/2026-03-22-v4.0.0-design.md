# v4.0.0 Design Spec — Morph, Gradient & Particle Categories + API Enhancements

**Date:** 2026-03-22
**Version:** 4.0.0
**Type:** Minor feature release (no breaking changes)

## Overview

v4.0.0 adds 3 new component categories (8 new components) and 3 non-breaking API enhancements to `premium-react-loaders`. The library grows from 50+ to 58+ components across 19 categories.

---

## New Component Categories

### 1. Morph (`src/components/morph/`)

Fluid, organic animation components based on CSS morphing and SVG path interpolation.

#### `MorphBlob`
An organic blob shape that continuously morphs between randomized border-radius states.

Props:
- `size?: number` — diameter in px (default: 80)
- `color?: string` — fill color (default: `#3b82f6`)
- `speed?: 'slow' | 'normal' | 'fast'` — morphing speed
- `complexity?: number` — number of morph keyframes (2–6, default: 4)
- Standard `BaseLoaderProps` (className, style, ariaLabel, testId, visible, reducedMotion)

#### `LiquidFill`
A circular or rectangular container that fills with an animated liquid level. Supports determinate (percentage-based) and indeterminate modes.

Props:
- `value?: number` — fill percentage 0–100 (indeterminate if omitted)
- `size?: number` — diameter in px (default: 80)
- `color?: string` — liquid color (default: `#3b82f6`)
- `waveAmplitude?: number` — wave height in px (default: 4)
- `onComplete?: () => void` — fires when value reaches 100
- Standard `BaseLoaderProps`

#### `WaveCircle`
Concentric circular ripples that emanate outward from the center, like a sonar pulse.

Props:
- `size?: number` — max ring diameter (default: 80)
- `color?: string` — ring color (default: `#3b82f6`)
- `ringCount?: number` — number of concentric rings (2–5, default: 3)
- `speed?: 'slow' | 'normal' | 'fast'`
- Standard `BaseLoaderProps`

---

### 2. Gradient (`src/components/gradient/`)

Visually striking spinners and progress indicators using animated CSS gradients.

#### `GradientSpinner`
A circular spinner filled with a sweeping conic gradient that rotates continuously.

Props:
- `size?: number` — diameter in px (default: 48)
- `colors?: [string, string]` — gradient start and end colors (default: `['#3b82f6', '#8b5cf6']`)
- `thickness?: number` — ring stroke width in px (default: 4)
- `speed?: 'slow' | 'normal' | 'fast'`
- Standard `BaseLoaderProps`

#### `GradientRing`
A ring/donut shape with a moving gradient that travels around its circumference.

Props:
- `size?: number` — diameter in px (default: 48)
- `colors?: string[]` — array of 2–4 gradient colors
- `thickness?: number` — ring width in px (default: 6)
- `speed?: 'slow' | 'normal' | 'fast'`
- Standard `BaseLoaderProps`

#### `GradientBar`
A horizontal progress bar with a flowing gradient shimmer animation.

Props:
- `value?: number` — progress 0–100 (indeterminate if omitted)
- `width?: number | string` — bar width (default: `100%`)
- `height?: number` — bar height in px (default: 8)
- `colors?: [string, string]` — gradient colors (default: `['#3b82f6', '#8b5cf6']`)
- `onComplete?: () => void` — fires when value reaches 100
- Standard `BaseLoaderProps`

---

### 3. Particle (`src/components/particle/`)

Particle-based loaders using canvas or CSS-positioned elements for dynamic visual effects.

#### `ParticleBurst`
Particles that periodically burst outward from a central point and fade as they travel, looping continuously.

Props:
- `size?: number` — burst radius in px (default: 60)
- `count?: number` — number of particles (4–20, default: 8)
- `color?: string` — particle color (default: `#3b82f6`)
- `particleSize?: number` — dot diameter in px (default: 6)
- `speed?: 'slow' | 'normal' | 'fast'`
- Standard `BaseLoaderProps`

#### `ParticleOrbit`
Multiple particles floating in circular orbits around a central dot, with staggered delays and optional trails.

Props:
- `size?: number` — orbital diameter (default: 80)
- `count?: number` — number of orbiting particles (3–8, default: 5)
- `color?: string` — particle color (default: `#3b82f6`)
- `centerColor?: string` — central dot color
- `showTrails?: boolean` — render fading trail behind each particle (default: false)
- `speed?: 'slow' | 'normal' | 'fast'`
- Standard `BaseLoaderProps`

## API Enhancements (Non-Breaking)

### 1. `onComplete` Callback

Added to components with a definable end state.

```ts
onComplete?: () => void
```

Applies to: `ProgressBar`, `ProgressCircle`, `ProgressRing`, `LiquidFill`, `GradientBar`

Fires when `value` prop reaches 100. Does not fire in indeterminate mode.

### 2. `reducedMotion` Prop

Added to `BaseLoaderProps` so all components inherit it.

```ts
reducedMotion?: boolean
```

When `true`, disables or simplifies animations (static or minimal opacity pulse). Complements the OS-level `prefers-reduced-motion` media query — useful when the developer needs programmatic control.

### 3. Size String Variants

Added to `BaseLoaderProps` alongside the existing numeric `size` prop.

```ts
size?: number | 'xs' | 'sm' | 'md' | 'lg' | 'xl'
```

Mapped values: `xs=24`, `sm=32`, `md=48`, `lg=64`, `xl=96`

Existing number-based usage is fully backward compatible.
